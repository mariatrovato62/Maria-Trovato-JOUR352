  4.  There are two main "areas" or "sections" in the basic structure of an HTML page. In which area would you put a link to a stylesheet?

  The two main areas are the head and the body. The head is the first part of the html page and contains general info about the html page. The body is the main part of the html page and contains the content to be displayed by the html browser. A link to a stylesheet would go in the head section.


5.  In which main "area" or "section" of an HTML page would you put text, images or DIV tags?

  10.  Why do we use media queries?

We use media queries to change the style of an element at different screen sizes. This is useful because content being viewed from a smartphone usually needs a different type of format to be more engaging and readable than content being viewed on a desktop or laptop.

  11. What is the main difference between a serif and a sans-serif font?

  A serif has feet, while a san-serif does not have feet. San-serif fonts are often used for headlines, while serif fonts is used more for body text. Serif examples include Georgia and Times. San-serif includes Helvetica and Arial.
